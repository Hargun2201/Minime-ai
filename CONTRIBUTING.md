# Contributing to MiniMe AI

Welcome! Thank you for your interest in contributing to MiniMe AI.

## Coding Style
- Follow **PEP 8** style guidelines.
- Use explicit type hints for all parameters and return types.
- Ensure all public-facing classes and methods have docstrings in Google style format.

## Development Workflow
1. Fork the repository and create your branch from `main`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Make changes and verify using `pytest tests/`.
4. Submit a Pull Request describing the changes.
